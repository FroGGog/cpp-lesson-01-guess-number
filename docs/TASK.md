# Урок 1: Угадай число

## Что мы сегодня изучим

- **Переменная** — именованная коробочка для хранения данных
- **Тип `int`** — целое число
- **Тип `bool`** — true или false
- **`std::cin`** — читает ввод с клавиатуры
- **`std::cout`** — выводит текст на экран
- **`if / else if / else`** — проверка условий
- **`while`** — цикл
- **`rand()` и `srand()`** — генератор случайных чисел

## Шпаргалка

```cpp
#include <iostream>  // для cout и cin
#include <cstdlib>   // для rand() и srand()
#include <ctime>     // для time()

int main()
{
    srand(time(0));  // включаем генератор случайных чисел
    
    int secret = rand() % 100 + 1;  // число от 1 до 100
    int guess = 0;
    
    std::cout << "Угадай число: ";
    std::cin >> guess;
    
    if (guess > secret) {
        std::cout << "Моё число меньше!" << std::endl;
    }
    
    return 0;
}
```

## Задание

Напиши игру "Угадай число":

1. Программа загадывает случайное число от 1 до 100
2. Игрок вводит свои догадки
3. После каждой догадки программа подсказывает:
   - "Моё число БОЛЬШЕ"
   - "Моё число МЕНЬШЕ"
4. Когда игрок угадал — вывести:
   ```
   Поздравляю! Ты угадал число 42 за 7 попыток!
   ```
5. После победы спросить:
   ```
   Сыграть ещё раз? (1 - да, 0 - нет):
   ```
   Если 1 — новая игра, если 0 — выход

## Подсказки

- `rand() % 100 + 1` даёт число от 1 до 100
- `srand(time(0))` пишется **один раз** в начале main
- Для цикла "пока не угадал": `while (guess != secret)`
- Для повтора всей игры: `while (playAgain)`
- Не забудь `#include <iostream>`, `<cstdlib>`, `<ctime>`
