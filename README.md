# Урок 1: Угадай число

Первый урок по C++. Учимся работать с переменными, вводом/выводом,
условиями и циклами на примере игры "Угадай число".

## Что нужно установить

1. **Visual Studio Community** (бесплатная)
   - https://visualstudio.microsoft.com/ru/vs/community/
   - При установке выбрать: "Разработка классических приложений на C++"

2. **Git**
   - https://git-scm.com/download/win
   - Установить с настройками по умолчанию

## Как работать

```bash
# 1. Клонируем репозиторий
git clone https://github.com/ТВОЙ_ЛОГИН/cpp-lesson-01-guess-number.git
cd cpp-lesson-01-guess-number

# 2. Создаём свою ветку
git checkout -b твоё_имя

# 3. Создаём папку
mkdir students\твоё_имя

# 4. Пишем код в файле students/твоё_имя/guess_number.cpp
```

## ✅ Как сдать задание

```bash
git add students/твоё_имя
git commit -m "Решение урока 1"
git push origin твоё_имя
```

Затем на GitHub создать Pull Request.
